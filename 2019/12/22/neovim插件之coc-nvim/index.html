安装注意事项

coc的补全是依赖语言自身的补全规则的，所以需要安装相应语言的language server。以python和c/c++举例：

python

python需要先安装pyls（python language server)并且需要正确配置到path系统环境配置中（划重点！！！）

否则会报以下的错误：

pyls not found in $PATH

c/c++

c/c++则需要先安装ccls(C/C++/ObjC language server)，严格的讲，它可以补全c, c++以及Object C三种语言。
